# Automatización para Android con Appium

Ejemplo básico de automatización de pruebas con Appium y TestNG

# Prerequisitos

Tener instalado Eclipse y el plugin de TestNG.
Tener instalado Appium.
Tener configuradas las variables de entorno JAVA_HOME y ANDROID_HOME

# Pasos para correr las pruebas

Correr el servidor de Appium en CMD
Correr los test desde la clase TestA
